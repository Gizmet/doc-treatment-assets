# Documentary Treatment Assets CDN

Image hosting for documentary treatment files via GitHub Pages.

## Structure

```
doc-treatment-assets/
├── toforest-johnson/     # Toforest Johnson case images
├── sam-hallam/           # Sam Hallam case images
├── gareth-jones/         # Gareth Jones case images
├── victor-nealon/        # Victor Nealon case images
└── sycamore-gap/         # Sycamore Gap case images
```

## CDN URLs

Images are available at:
```
https://gizmet.github.io/doc-treatment-assets/{case-name}/{image-file}
```

## Usage

Treatment HTML files reference images via absolute CDN URLs for reliability across all devices (desktop, mobile, email, AirDrop).
